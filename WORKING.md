## 使用协议

### 开始工作
1. 读此文件，确认「正在干活」为空或自己的任务不冲突
2. 写上自己的行: `| Claude Code | 修XXX bug | 14:30 | 15:00 |`
3. **git add WORKING.md && git commit -m "checkin: Claude Code working on XXX"**

### 完成工作
1. **更新 CLAUDE.md 底部的 [变更记录]**
2. 清除此文件中自己的行
3. **git add -A && git commit -m "<AI>: <做了什么>"**
4. 如果还有在等的 AI，告诉它：`请看 CLAUDE.md 最新变更记录`

### 接手别人的工作
1. 读 CLAUDE.md 的 [变更记录]
2. 读 `git log --oneline -5`
3. 确认上一个 AI 已完成并提交
4. 按「开始工作」流程办理
